This application is used to introduce 5 games published by Ubisoft.

Games included:
Prince of Persia
Tom Clancy's Splinter Cell
Assassin Creed
Far Cry
Watch Dogs

The user can access to the introduction page by simply clicking on the game name. If more details are needed, the user can click on the more details for more details. Moreover, they can back to the previous page by clicking "Back" button.
